# Aligned Aminos — Website

Static two-page site for the Aligned Aminos peptide brand. No build step. No framework. Just HTML and inline CSS/JS.

Auto-deploys to Vercel on every push to `main`.

## Files

- `index.html` — Home page
- `catalog.html` — Product catalog (prices live inside this file; see `HOW_TO_UPDATE.md`)
- `datasheets/` — Standalone RUO data sheets for each SKU
- `logo.png`, `logo.jpg` — Logo assets used in the nav

## To update the catalog

See `HOW_TO_UPDATE.md`.

## Compliance

All product content is research-use-only (RUO). Do not introduce copy that implies human consumption, dosing, or medical claims.
